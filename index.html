Navegadores web

Un navegador web (en inglés, web browser) es un software, aplicación o prgrama que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados.

Ejemplos de navegadores web

Safari: es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS
Mozilla Firefox: es un navegador web libre y de código abierto desarrollado por Linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla.
Google Chrome: es un navegador web de código cerrado desarrollado por Google aunque ha derivado de proyectos de código abierto.
